# ☕ Coffee News システム構成

現在の「Coffee News」アプリケーションのシステム構成です。

## 1. プロジェクト概要

世界中のコーヒーに関する最新ニュースを自動で収集し、AI（Gemini）が日本語に翻訳・要約して表示するWebサイトです。

## 2. システム構成図

```text
┌───────────────────┐      ┌────────────────────┐      ┌──────────────────────┐
│   ユーザーのブラウザ   │      │ Frontend (Next.js) │      │ Database (Firestore) │
│ (PC/スマホ)       ├─────▶│                    ├─────▶│       (articles)     │
└───────────────────┘      └─────────┬──────────┘      └─────────▲────────────┘
                                     │                          │
                               (Reads Data)                   (Writes Data)
                                     │                          │
                                     ▼                          │
┌───────────────────┐      ┌────────────────────┐      ┌──────────────────────┐
│ Scheduler         │      │ Backend            │      │ AI (Gemini API)      │
│ (Cloud Scheduler) ├─────▶│ (Cloud Functions)  ├─────▶│                      │
│ (毎日朝4時に実行)    │      │ (main.py)          │      └──────────────────────┘
└───────────────────┘      └─────────┬──────────┘
                                     │
                               (Fetches RSS)
                                     │

                             ┌────────────────────┐
